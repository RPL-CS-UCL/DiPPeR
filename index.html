DiPPeR: Diffusion-based 2D Path Planner applied on Legged Robots

Jianwei Liu*             Maria Stamatopoulou*             Dimitrios Kanoulas



Abstract

In this work, we present DiPPeR, a novel and fast 2D path planning framework for quadrupedal locomotion, leveraging diffusion-driven techniques. Our contributions include a scalable dataset of map images and corresponding end-to-end trajectories, an image-conditioned diffusion planner for mobile robots, and a training/inference pipeline employing CNNs. We validate our approach in several mazes, as well as in real-world deployment scenarios on Boston Dynamic's Spot and Unitree's Go1 robots. DiPPeR performs on average 70 times faster for trajectory generation against both search based and data driven path planning algorithms with an average of 80% consistency in producing feasible paths of various lengths in maps of variable size, and obstacle structure.


In the following sections we present DiPPeRs performance and genarilisation capabilities.

Validation Dataset


We validate DiPPeR's performance on maps on the Validation Datase. It contains maps which were not seen during training but have a similar structure to the Training Dataset ones.

Out-of-Distribution Dataset


We evaluate DiPPeR's generalisation cabailities in maps different than the Training Dataset ones.


OOD Dataset

Dataset consist of maps of different size, obstacle structure and colour, to test the generalisation capabilities of DiPPeR.

MRPB Dataset

Dataset used to compare DiPPeR inference speed against A* and N-A*. DiPPeR's inference time is 0.4s for all maps despite their size, which is one average 70 times faster than the SOTA planners.




Real World Deployment

DiPPeR is integrated with the 2D ROS navigation Stack, to acts as a global path planner. Given the occupancy map, DiPPeR generates a global path which is further refined by the TEB local planner - to avoid violation of the robots kinodyanmics constraints and an Phasespace Localisation system - to mitigate for state estimation inaccuracies.
